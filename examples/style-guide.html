CSS Variables Reference

All framework values exposed as CSS custom properties for maximum flexibility.

Color Variables

/* Brand Colors */
--primary, --primary-light, --primary-dark
--secondary, --secondary-hover
--accent, --accent-ultra-light
--base, --shade, --neutral

/* Transparency Variants */
--primary-trans-10 through --primary-trans-90

/* Contextual Colors */
--success, --warning, --danger, --info

Typography Variables

/* Headings */
--h1, --h2, --h3, --h4, --h5, --h6

/* Text Sizes */
--text-xs, --text-s, --text-m
--text-l, --text-xl, --text-xxl

/* Font Families */
--font-primary, --font-secondary, --font-mono

/* Font Weights */
--font-weight-light through --font-weight-bold

Spacing Variables

/* Component Spacing */
--space-xs, --space-s, --space-m
--space-l, --space-xl, --space-xxl

/* Contextual Gaps */
--content-gap, --grid-gap, --container-gap

/* Section Spacing */
--section-space-s through --section-space-xxl

Layout Variables

/* Grid Templates */
--grid-1 through --grid-12
--grid-auto-1 through --grid-auto-6

/* Container Widths */
--content-width, --content-width-safe

/* Aspect Ratios */
--aspect-square, --aspect-video
--aspect-photo, --aspect-golden

Effects Variables

/* Shadows */
--shadow-s, --shadow-m, --shadow-l, --shadow-xl
--shadow-primary, --shadow-accent

/* Border Radius */
--radius-s, --radius-m, --radius-l, --radius-xl

/* Transitions */
--transition, --transition-fast, --transition-slow

Usage Examples

/* Direct Usage */
background: var(--primary-light);
padding: var(--space-l);
font-size: var(--text-xl);

/* With calc() */
gap: calc(var(--grid-gap) * 2);
margin: calc(var(--space-m) * -1);

/* Fallbacks */
color: var(--primary, #0066cc);
