Typography Variables

The framework exposes all typography settings as CSS variables for maximum flexibility:

--h1 through --h6 (heading sizes)

--text-xs through --text-xxl (t-shirt sizes)

--font-primary, --font-secondary, --font-mono

--font-weight-light through --font-weight-bold

--text-line-height, --heading-line-height

--text-letter-spacing, --heading-letter-spacing
